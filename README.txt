# ReinadoRPG Site (GitHub Pages)

## Como usar
1. Faça upload de tudo deste zip para um repositório (raiz).
2. GitHub -> Settings -> Pages -> Deploy from a branch -> main /(root)
3. Pronto.

## Configurar links
Abra `app.js` e edite:
- `whatsappCommunity`: cole o link da sua comunidade/canal.
- `serverAddress`: IP/domínio do servidor (se mudar).

## API de players online (sem mostrar versão)
